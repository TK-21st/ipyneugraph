
ipyneugraph
=====================================

Version: |release|

An IPyWidget extension for handling NeuroDriver-compatible computational graphs.


Quickstart
----------

To get started with ipyneugraph, install with pip::

    pip install ipyneugraph

or with conda::

    conda install ipyneugraph
